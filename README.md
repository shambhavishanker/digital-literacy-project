## Project Overview
This project was completed as part of the Digital Literacy course, where I took on the role of a Student Digital Ambassador. The aim of this project was to understand how digital tools are used in real world situations and how to use them responsibly and effectively.
The project consists of multiple tasks that focus on different aspects of digital literacy, including creating awareness content, building an online professional presence, practicing coding skills, understanding communication etiquette, and learning about cybercrime and online safety.
Through these tasks, I was able to apply theoretical knowledge in a practical way and improve my understanding of how digital platforms impact both academic and professional environments.

TASKS:
### Task 1: Digital Literacy Infographic
In this task, I created an infographic to spread awareness about digital literacy using Canva. The focus was on explaining key concepts such as online presence, safety, critical thinking, and responsible use of technology in a simple and visually clear way.
